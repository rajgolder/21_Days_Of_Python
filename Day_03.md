# 📘 Day 3 Revision Notes

## Boolean

- Boolean values represent only two states:
  - `True`
  - `False`
- Used mainly in comparisons and conditional statements.
- Python is case-sensitive: use `True` and `False` (not `true` or `false`). :contentReference[oaicite:0]{index=0}

---

# Operators

Operators perform operations on values and variables.

## 1. Assignment Operators

Used to assign values to variables.

Common operators:
- `=`
- `+=`
- `-=`
- `*=`
- `/=`
- `//=`
- `%=`
- `**=`

**Remember:** `=` means **assignment**, not equality. :contentReference[oaicite:1]{index=1}

---

## 2. Arithmetic Operators

| Operator | Purpose |
|----------|---------|
| `+` | Addition |
| `-` | Subtraction |
| `*` | Multiplication |
| `/` | Division (returns float) |
| `//` | Floor Division |
| `%` | Modulus (remainder) |
| `**` | Exponent (power) |

### Notes
- `/` always returns a float.
- `//` removes the decimal part.
- `%` gives the remainder.
- `**` calculates powers. :contentReference[oaicite:2]{index=2}

# Comparison Operators

Used to compare two values.

| Operator | Meaning |
|----------|---------|
| `==` | Equal to |
| `!=` | Not equal to |
| `>` | Greater than |
| `<` | Less than |
| `>=` | Greater than or equal |
| `<=` | Less than or equal |

### Result
Comparison operators always return:
- `True`
- `False` :contentReference[oaicite:3]{index=3}

# Identity & Membership Operators

## Identity Operators

- `is` → Same object
- `is not` → Different objects

## Membership Operators

- `in` → Item exists
- `not in` → Item does not exist :contentReference[oaicite:4]{index=4}

---

# Logical Operators

Used to combine multiple conditions.

| Operator | Meaning |
|----------|---------|
| `and` | True if both conditions are True |
| `or` | True if at least one condition is True |
| `not` | Reverses the boolean value |

### Truth Rules

- `True and True` → True
- `True and False` → False
- `True or False` → True
- `False or False` → False
- `not True` → False
- `not False` → True :contentReference[oaicite:5]{index=5}

---

# Important Points to Remember

- Boolean values are only `True` and `False`.
- Arithmetic operators perform calculations.
- Comparison operators return boolean values.
- `=` assigns a value.
- `==` checks equality.
- `/` gives a float.
- `//` gives integer (floor) division.
- `%` returns the remainder.
- `**` means exponent (power).
- `is` checks object identity.
- `in` checks membership.
- `and`, `or`, and `not` are logical operators.
